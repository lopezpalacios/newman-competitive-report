# Inteligencia Competitiva · Energía C&I México 2026

Reporte de investigación competitiva del sector de energía distribuida comercial e industrial (C&I) en México, por Newman Energy. Analiza a Energía Real, Enlight, Niko y Quartux a través de seis frentes: capacidad verificable, marca/persuasión, SEO, precios, capital y exposición a la reforma eléctrica 2024–2026.

**Sitio:** página estática (HTML/CSS/JS vanilla, sin dependencias) lista para GitHub Pages.

## Publicar en GitHub Pages
1. Crear repo en GitHub y hacer push de esta carpeta.
2. Settings → Pages → Source: `Deploy from a branch` → `main` / `/ (root)`.
3. El sitio queda en `https://<usuario>.github.io/<repo>/`.

## Archivos
- `index.html` — el reporte (scrollytelling, gráficos animados, mapa de capital SVG, matriz de exposición).
- `newman.css` / `newman.js` — kit de marca Newman (gradiente navy→magenta, count-up, reveal, barras animadas).
